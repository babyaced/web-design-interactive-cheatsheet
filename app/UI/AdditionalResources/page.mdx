# Additional Resources

## Inspiration
- [Mobbin](https://mobbin.com/) - Collection of screenshots from mobile apps showing design patterns.
- [Component Gallery](https://component.gallery/) - A gallery of UI components from popular websites.
- [Design Systems for Figma](https://www.designsystemsforfigma.com/) - A collection of free design systems and UI kits for Figma.
- [Swype](https://swype.design/) - A completely free collection of mobile website screenshots for design inspiration.
- [Refero](https://refero.design/) - A collection of desktop and mobile website screenshots for design inspiration and UI/UX patterns.
- [Awwwards](https://www.awwwards.com/collections/) - A website that recognizes and promotes the talent and effort of the best web designers, developers, and agencies in the world.

## Tools
- [Realtime Colors](https://realtimecolors.com/) - A tool for generating and visualizing color palettes (and fonts!)
- [FontPair](https://fontpair.co/) - Generate font pairings and visualize them with sample text.
- [WAVE](https://wave.webaim.org/) - A web accessibility evaluation tool that finds accessibility issues on your website.
